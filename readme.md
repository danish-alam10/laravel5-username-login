# Project Title

Login with Email and Username in laravel 5.6

### Prerequisites

Basic knowledge of Laravel 5

### Installing

1. Clone the project in your local directory
2. Run command 
    ``` composer install ```
3. Setup database
4. Start local development environment with
    ``` php artisan serve ```

You are all set!
